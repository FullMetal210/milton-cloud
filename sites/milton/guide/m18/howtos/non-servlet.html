Milton can easily be integrated with non servlet-api web and desktop applications. To do you just need to  implement your own com.bradmcevoy.http.Request and Response

  1. At startup your application should create an instance of ResourceFactory
  2. Then create an instance of HttpManager, giving it your resource factory
  3. For each request that your application receives, create instances of Request and Response
  4. Then pass the request and response to the process method of HttpManager

For an example, please see the Berry project . This project integrates Milton with xLightWeb and Simple HTTP embedded http servers.
