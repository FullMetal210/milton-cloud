Once the Milton AJAX gateway is hooked up to your application you can query files and folders using AJAX by appending commands to the URL.

The general syntax is:

{physical url}/_DAV/{dav method}?{parameters}

For browsing the dav method is PROPFIND which takes a "fields" parameter. This parameter is a comma seperated list of properties as defined by webdav.

For example, to get the contents of this folder you could call this url:

http://milton.ettrema.com/guide/modules/milton-json/_DAV/PROPFIND?fields=name,getcontenttype

You can click that link and try it out.
