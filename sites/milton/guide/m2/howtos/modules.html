Milton consists of a core API library and a bunch of other modules

Module Type Notes
milton-api jar

Core library containing Resource interfaces and protocol handler implementations. Is independent of the servlet API

You definitely want this.
milton-servlet jar Contains adapters for normal servlet containers. You probably want this.
milton-test war

A runnable web application which demonstrates implementing milton resource interfaces in the simplest manner.

Uses memory only data storage so is not not suitable for production.

Use it to learn how to implement milton resources.
milton-filesystem jar

Resource implementation for using a physical filesystem as a webdav repository.

Use this if you want an off-the-shelf file server, or if you want to use the code as an example
milton-filesystem-test war

A complete web application which demonstrates using milton-filesystem. Uses spring configuration.

Uses very simple user management which might not be suitable for production, but is otherwise a useful webdav server.
milton-gae jar

Just some helper class implementations to  allow milton to be used in Google App Engine.

Use it if you're deploying into GAE
milton-json jar

Adapters to allow WEBDAV operations using GET and POST through javascript.

This allows a web developer to leverage the webdav capabilities of the server in rich web 2.0 web apps. For example you can call PROPFIND and get the result as a JSON array.
milton-ajax-demo war Demonstrates using milton-json.
milton-ftp jar

Adapter to Apache FTP, allows milton resources to be accessed by FTP.

Use it if you want to access your data by both FTP and webdav
milton-console jar

Fun little addition. It gives you a command line console in a web page which you can use to navigate your webdav repository.

Use it to look cool.
milton-caldav jar

Experimental implementation of caldav and ACL.Should be finished late 2010. Currently works with mozilla calendar clients, and is being extended to include iCal

Use it to turn your milton server into a CALDAV server.
milton-caldav-demo war Demonstrates using milton-caldav
