You can include the milton jars in your maven pom.xml file like this:

        <dependency>
            <groupId>io.milton</groupId>
            <artifactId>milton-server</artifactId>
            <version>${milton.version}</version>
        </dependency>    

However, milton is not published to the central repositories, so you'll need to add the ettrema repository:

    <repositories>
        <repository>
            <id>ettrema-repo</id>
            <url>http://www.ettrema.com/maven2</url>
        </repository>
    </repositories>
