Milton supports Caldav via an extension module. To use it, you need to:

  1. wire up the extension into Milton
  2. implement the caldav interfaces
  3. run the server
  4. point iCal or your favourite caldav client at the right location

There are two example projects in subversion

  • milton-caldav-demo - based on maven and spring, uses an in memory implementation
  • examples/caldav-non-spring-netbeans - does not use maven or spring, just a plain old netbeans project
