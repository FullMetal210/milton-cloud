Here, we will start with a simple dirty program before we go into the details of Milton. Milton provides all of the WebDAV functionalities and very customizable according to your need. Following program just lets you get started on a simple program. The main objective of the following code is to get your hand dirty with Milton API. Hence it is simplified as much as possible. If you are already familiar with Milton, you can skip this section.

Implement the Resource (like GetableResource, FileResource etc): If you like to download the say some document, then need to implement the GetableResource and write a logic to send the content in the SendContent method.
